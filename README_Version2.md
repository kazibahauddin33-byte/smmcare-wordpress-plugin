# SMMCARE WordPress Plugin

SMMCARE integration plugin (initial scaffold).

## Installation

1. Clone the repo into `wp-content/plugins/smmcare` or install via zip.
2. Activate the plugin from WordPress admin.
3. Visit SMMCARE -> Settings to configure the plugin.

## Development

- PHP / WordPress environment recommended: Local WP / Docker / VVV.
- Run linting: `php -l includes/*.php smmcare.php`
- Unit tests: set up PHPUnit with WP testing framework (optional).
